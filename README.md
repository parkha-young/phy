저희는 오픈 API를 사용하였습니다. 
오픈 API란 누구나 사용할 수 있도록 공개된 응용프로그램 개발환경을 의미하며, 공공기관에서 프로그램의 기능을 공개적으로 배부하여 사용자가 또 다른 프로그램에서 활용할 수 있도록 하는 것입니다.
첫 번째로 깃허브의 끝말잇기 오픈 API를 활용하였으며, 1인용 게임에서 더 나아가 2인용 게임을 추가하였습니다.
두 번째로 국립국어원 오픈API의 기초국어대사전이 아닌 표준국어대사전 오픈 API를 사용했으며, 검색 요청변수를 통해 저희가 구하고자 하는 단어들을 웹크롤링 하지 않고도 구할 수 있도록 하였습니다. 
기초국어대사전의 경우 단어의 수가 적고 제한적이었기 때문에 표준 국어 대사전의 API를 사용하였습니다.
사용한 모듈은 Requests, hgtk, random 함수입니다.
